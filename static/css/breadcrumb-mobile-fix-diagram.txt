CSS PROPERTIES APPLIED:
-----------------------

1. Touch Target Size:
   min-height: 44px (desktop) / 48px (mobile)
   min-width: 44px (desktop) / 48px (mobile)

2. Clickable Area Expansion:
   padding: 0.5rem
   margin: -0.5rem  ← Expands clickable area

3. Touch Optimization:
   touch-action: manipulation  ← Removes 300ms delay
   -webkit-tap-highlight-color: rgba(236, 19, 19, 0.2)

4. Z-Index Management:
   position: relative
   z-index: 100  ← Always on top

5. Prevent Icon Interference:
   .material-symbols-outlined {
     pointer-events: none  ← Icons don't block clicks
     user-select: none
   }
